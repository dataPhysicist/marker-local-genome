# Security & privacy

## What stays local

- Raw genotype files are parsed and interpreted **only inside your browser** (web build) or inside the **Marker desktop app** on your Mac. They are **not uploaded** to Marker’s servers — this project does not operate a backend for user data.

## Optional network use

- If you use **optional enrichment**, the app requests **public variant metadata by rsID only** (e.g. MyVariant.info). Your file contents and genotypes are not sent in those requests.

## Reporting issues

If you discover a security problem (e.g. accidental data exfiltration in the client), please open a **private** security advisory on GitHub or contact the repository maintainers.
